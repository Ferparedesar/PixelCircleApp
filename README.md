# Pixel Circle Animation

Este proyecto es una aplicación web en Java que muestra un píxel girando en círculos mediante un lienzo HTML5. 

## Cómo ejecutar
1. Clona este repositorio.
2. Usa `mvn clean package` para empaquetar el proyecto.
3. Despliega el archivo `.war` en un servidor compatible, como Tomcat.

## URL del Servlet
Accede a la animación en `/PixelCircle`.
